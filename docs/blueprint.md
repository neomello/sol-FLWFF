# **App Name**: FLWFF Stablecoin Info

## Core Features:

- Price Display: Integração com CoinGecko API pra puxar o preço real-time do token `$FLWFF`. Visualização em destaque, fonte monoespaçada, com efeito neon/verde ácido no valor.
- Whitelist Form: Formulário com campos de wallet (ETH / SOL) e e-mail. Dados serão armazenados via Firebase Firestore. Estilo agressivo com cores do projeto e tom de seita/convocação.
- IPFS Export: Ao enviar o form, os dados são empacotados e exportados para IPFS. Registro imutável das almas coletadas — com hash visível e botão pra visualizar no gateway.
